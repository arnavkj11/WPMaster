## ✨ Features

### 🎯 Core Functionality

- **Real-time WPM Calculation** - Accurate words-per-minute tracking using the standard 5-character word formula
- **Live Accuracy Metrics** - Instant feedback on typing accuracy with error highlighting
- **Multiple Test Durations** - Choose from 30s, 60s, or 120s typing tests
- **Character-by-Character Tracking** - Visual feedback with color-coded characters (green for correct, red for errors)

### 🤖 AI-Powered Features

- **AI Text Generation** - Custom typing passages generated by Claude 3.7 Sonnet based on:
  - Difficulty level (beginner, intermediate, advanced)
  - Category preference (general, programming, quotes)
  - Your past typing mistakes for targeted practice
- **Personalized Coaching** - AI-powered error analysis with actionable improvement tips
- **Adaptive Learning** - Texts that adapt to your skill level and weak areas

### 📊 Advanced Analytics

- **WPM Progression Chart** - Real-time graph showing your typing speed over time
- **Keyboard Heatmap** - Visual representation of key press frequency and accuracy
- **Error Pattern Analysis** - Identify which character combinations you struggle with most
- **Common Mistakes Tracking** - See your most frequent typing errors
- **Peak Performance Metrics** - Track your highest WPM and accuracy scores

### 🎨 User Experience

- **Dark/Light Theme** - Toggle between themes with persistent preference storage
- **Typing Sound Effects** - Optional audio feedback for keystrokes (can be toggled)
- **Auto-Scrolling Text** - Long texts automatically scroll to keep your position visible
- **Responsive Design** - Works seamlessly on desktop and tablet devices
- **Smooth Animations** - Polished UI with Tailwind CSS transitions

### 👤 User Accounts & Progress

- **User Authentication** - Secure JWT-based login and registration
- **Progress Tracking** - All your test results are saved and tracked over time
- **Personal Statistics** - Detailed dashboard showing your improvement journey
- **Global Leaderboard** - Compare your scores with other users

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- MongoDB Atlas account or local MongoDB installation
- Anthropic API key ([Get one here](https://console.anthropic.com/settings/keys))

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/yourusername/typing-speed-test-app.git
cd typing-speed-test-app
```

2. **Install backend dependencies**

```bash
cd server
npm install
```

3. **Install frontend dependencies**

```bash
cd ../client
npm install
```

4. **Configure environment variables**

Create `server/.env`:

```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/?retryWrites=true&w=majority
PORT=5000
JWT_SECRET=your_secure_jwt_secret_key_here
ANTHROPIC_API_KEY=your_anthropic_api_key_here
```

Create `client/.env`:

```env
VITE_API_URL=http://localhost:5000/api
```

5. **Start the development servers**

Backend (from `server/` directory):

```bash
npm run dev
```

Frontend (from `client/` directory):

```bash
npm run dev
```

6. **Access the application**

- Frontend: http://localhost:5173
- Backend API: http://localhost:5000

## 📖 Usage

### Taking a Typing Test

1. Navigate to the test page
2. Select your preferred:
   - **Difficulty level** (beginner/intermediate/advanced)
   - **Category** (general/programming/quotes)
   - **Duration** (30s/60s/120s)
3. Optionally click **"Generate AI Text"** for custom content
4. Start typing in the input field
5. View your results with detailed analytics

### Viewing Results

After completing a test, you'll see:

- Your WPM score and performance rating
- Real-time accuracy percentage
- Peak WPM achieved during the test
- WPM progression chart
- Common mistakes analysis
- Keyboard heatmap showing key accuracy
- Optional AI coaching feedback

### Creating an Account

1. Click **"Login / Register"**
2. Create an account or log in
3. All your test results will be automatically saved
4. Access your **"My Stats"** page for historical data

## 🎯 API Endpoints

### Authentication

```
POST   /api/auth/register    - Create new user account
POST   /api/auth/login       - Login and receive JWT token
GET    /api/auth/profile     - Get current user profile (Protected)
```

### Test Results

```
POST   /api/results                        - Save test result (Protected)
GET    /api/results/user?page=1&limit=10   - Get user's test history (Protected)
GET    /api/results/stats                  - Get user statistics (Protected)
GET    /api/results/leaderboard?duration=60&limit=10 - Get leaderboard (Public)
```

### AI Features

```
POST   /api/ai/generate-text   - Generate custom typing text
POST   /api/ai/analyze-errors  - Get personalized error analysis
```

## 🐛 Known Issues & Troubleshooting

### NaN Display Issues

If you see "NaN" in WPM or time displays, ensure all state variables are properly initialized. The latest version includes comprehensive NaN protection.

### AI Features Not Working

- Verify your `ANTHROPIC_API_KEY` is set in `server/.env`
- Restart the backend server after adding environment variables
- Check server console for detailed error messages

### Text Not Auto-Scrolling

- Ensure the text display has the `typing-text-scroll` class
- Verify refs are properly attached to DOM elements
